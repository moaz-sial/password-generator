# Random Password Generator

This Python script allows you to generate a random password by specifying the number of special characters, lowercase letters, and uppercase letters.

## Usage

1. Ensure you have Python 3 installed.
2. Run the script:
    ```bash
    python3 random_password_generator.py
    ```
3. Follow the prompts to input the desired number of special characters, lowercase letters, and uppercase letters.

## Example

```bash
python3 random_password_generator.py
